### 3. 创建数据库集合
云开发控制台 → 数据库 → 新建集合：

| 集合名 | 权限 | 用途 |
|--------|------|------|
| `User` | 所有用户可读，仅创建者可写 | 用户信息 |
| `Road` | 所有用户可读，仅创建者可写 | 路段数据 |
| `Ticket` | 所有用户可读，仅创建者可写 | 罚单/评论 |
| `DailyLimit` | 仅创建者可读写 | 每日限制 |

### 4. 创建数据库索引
云开发控制台 → 数据库 → 对应集合 → 索引管理 → 新建：

| 集合 | 索引字段 | 方向 |
|------|----------|------|
| `Ticket` | `roadId` + `createdAt` | 升序 + 降序 |
| `Ticket` | `userId` + `createdAt` | 升序 + 降序 |
| `Road` | `totalTickets` | 降序 |

- [x] 去 [lbs.qq.com](https://lbs.qq.com) 创建 Key，勾选 WebService API → 选择「授权 IP」→ 填 `0.0.0.0/0`
